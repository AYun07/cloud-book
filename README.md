## 核心特性

### 🤖 全模型支持

- **OpenAI 系列**: GPT-4, GPT-3.5-turbo, 及所有兼容API的模型
- **Claude 系列**: Claude 3 Opus, Sonnet, Haiku
- **DeepSeek 系列**: DeepSeek Chat, DeepSeek Coder
- **本地部署**: Ollama, KoboldCPP, LM Studio, vLLM
- **智能路由**: 根据任务类型自动选择最优模型
- **离线支持**: 内置API代理服务器，离线也能调用云端API

### 🌍 全球语言支持

- **40+种语言**: 中文(简/繁)、英语、日语、韩语、法语、德语、西班牙语、俄语、阿拉伯语、印地语等

- **拼写检查**: 各语言词典支持
- **本地化格式**: 数字、日期、货币、列表格式自动适配
- **语言自动检测**: 智能识别输入语言

### ✨ 核心能力

#### 百万字长篇处理
- **智能解析**: 自动拆解分析数百万字原作
- **风格指纹**: 精准提取原文写作风格特征
- **分块处理**: 智能分章节处理超大文本
- **一致性保障**: 跨章节角色和世界观一致性维护

#### 一键创作模式
- **原创模式**: 从零开始构建全新故事
- **仿写模式**: 保持风格的高仿创作
- **二创模式**: 基于原作的创新改编
- **同人模式**: 保留角色设定的新故事

#### 智能创作管线
- **六类创作智能体**:
  - 架构师: 世界观构建、角色设定、情节规划
  - 写作者: 具体章节写作生成
  - 审计员: 33维度质量检查
  - 修订师: 问题修复和质量优化
  - 风格工程师: 文风把控和人性化处理
  - 雷达: 趋势分析和市场洞察

- **七步创作法**:
  1. 宪章制定 - 确定创作原则
  2. 规格说明 - 明确故事规格
  3. 细节澄清 - 完善世界观设定
  4. 规划制定 - 章节规划和里程碑
  5. 任务分解 - 具体任务拆分
  6. 写作执行 - 分步创作实施
  7. 分析复盘 - 结果分析和优化

#### 33维度质量审计
| 维度类别 | 具体项目 |
|----------|----------|
| 一致性检查 | 角色一致性、世界观一致性、时间线一致性 |
| 逻辑检查 | 情节逻辑、伏笔呼应、资源追踪 |
| 节奏检查 | 叙事节奏、进度节奏、冲突升级 |
| 角色检查 | 角色动机、角色发展、对话质量 |
| 描写检查 | 描写密度、感官细节、展示vs讲述 |
| AI检测 | 33+维度AI特征识别、人机交互处理 |
| 其他检查 | 重复性、语法错误、同义反复、逻辑漏洞、悬念清晰度、背景整合、视角一致、时态一致、节奏变化、潜台词、象征主义、主题连贯、读者参与、类型惯例、文化敏感、事实准确 |

- **记忆系统**: 记忆/作者笔记/系统提示分类管理

## 快速开始

```typescript
import { CloudBook } from '@cloud-book/core';

const cloudBook = new CloudBook({
  llmConfigs: [
    {
      provider: 'openai',
      name: 'gpt-4',
      apiKey: process.env.OPENAI_API_KEY,
      model: 'gpt-4'
    },
    {
      provider: 'anthropic',
      name: 'claude-3',
      apiKey: process.env.ANTHROPIC_API_KEY,
      model: 'claude-3-sonnet-20240229'
    }
  ],
  modelRoutes: [
    { task: 'planning', llmConfig: { name: 'claude-3' } },
    { task: 'writing', llmConfig: { name: 'gpt-4' } },
    { task: 'audit', llmConfig: { name: 'claude-3' } }
  ],
  i18nConfig: {
    primaryLanguage: 'zh-CN',
    fallbackLanguage: 'en-US',
    autoDetect: true,
    grammarCheck: true,
    spellCheck: true
  },
  connectionMode: 'hybrid',
  auditConfig: {
    enabled: true,
    dimensions: ['characterConsistency', 'plotLogic', 'aiDetection']
  },
  antiDetectionConfig: {
    enabled: true,
    intensity: 0.7,
    replaceAIWords: true,
    varySentenceStructure: true
  }
});

// 创建项目 - 支持所有题材
const project = await cloudBook.createProject(
  'My Novel',
  'fantasy',  // 题材
  'novel'     // 体裁
);

// 生成章节
const chapter = await cloudBook.generateChapter(project.id, 1, {
  autoAudit: true,
  autoHumanize: true
});

// 批量生成
const chapters = await cloudBook.batchGenerateChapters(project.id, 2, 10, {
  parallelCount: 3,
  autoAudit: true
});
```

## 功能示例

### 多语言支持

```typescript
// 设置语言
cloudBook.setLanguage('en-US');

// 检测语言
const detected = cloudBook.detectLanguage('Hello world');

// 语法检查
const grammarCheck = await cloudBook.checkGrammar('This is a test sentence.');
```

### 离线API调用

```typescript
// 启动本地API服务器
await cloudBook.startLocalAPIServer({
  port: 8080,
  apiKeys: [
    {
      provider: 'openai',
      apiKey: 'your-api-key',
      models: ['gpt-4', 'gpt-3.5-turbo']
    }
  ]
});

// 离线模式调用
const result = await cloudBook.completeOffline('Write a story...');
```

### 世界构建

```typescript
// 添加世界设定
await cloudBook.addWorldInfo(project.id, {
  name: 'Magic System',
  keywords: ['magic', 'spell', 'casting'],
  content: 'The magic in this world is based on elemental forces...',
  depth: 1
});

// 构建角色关系图谱
await cloudBook.addCharacterToGraph(project.id, {
  name: 'Hero',
  personality: 'Brave and determined',
  background: 'A young orphan with hidden powers'
});
```

### 创作流程

```typescript
// 执行七步创作法
await cloudBook.executeMethodologyStep(project.id, 'constitution', {
  genre: 'fantasy'
});

// 使用六类智能体
const agents = cloudBook.getAgents();
const result = await cloudBook.executePipeline(project, chapterNumber);
```
