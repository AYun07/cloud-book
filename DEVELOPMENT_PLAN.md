# Cloud Book 详细开发计划

## 项目信息

- **项目名称**: Cloud Book - AI小说创作平台
- **项目地址**: https://github.com/AYun07/cloud-book
- **技术栈**: TypeScript Monorepo
- **目标**: 完全国产原创的AI小说创作平台，整合全球优秀创作工具核心优势
